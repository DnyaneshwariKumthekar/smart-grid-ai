
# SMART GRID AI - USER GUIDE FOR OPERATORS

## Quick Start (5 minutes)
1. Open Dashboard: http://localhost:8501
2. Navigate to "Forecasts" tab
3. View 7-day ahead predictions
4. Check anomaly alerts in red

## Understanding the Dashboard
- **Metric Cards**: Key performance indicators updated hourly
- **Forecast Chart**: Point forecast with 95% confidence bands
- **Anomaly Panel**: Real-time anomalies with severity levels
- **Business Impact**: ROI and savings calculations

## Interpreting Predictions
- **Green Band**: 95% confidence interval
- **Blue Line**: Most likely forecast
- **Red Points**: Detected anomalies requiring investigation

## Common Questions
Q: Why is confidence interval so wide?
A: Wide intervals indicate high uncertainty, often due to weather or external events.

Q: How often are models retrained?
A: Daily at 2:00 AM UTC using fresh data.

Q: What should I do if I see anomalies?
A: 1) Check the root cause suggestion
   2) Review recent events/maintenance
   3) Contact support if unexplained
